COBOL Analysis and Refactoring Webapp

Main functionality:
  • Cobol Reference Grammar 
  • Cobol Source Analysis     (part of project)
  • Cobol Source Editing       (part of project)
  • Cobol Source Search       (part of project)
  • Cobol Source Refactoring (part of project)
URLs:
  • oncobol.anubex.com/about -> main intro page with links to 'grammar', 'faq', 'registration' and 'login'
  • oncobol.anubex.com/register
  • oncobol.anubex.com/grammar 
  • oncobol.anubex.com/faq
  • oncobol.anubex.com/login
  • oncobol.anubex.com/project -> main cobol project page where analysis, editing, search and refactoring can happen
  • oncobol.anubex.com/settings -> per user settings
  • oncobol.anubex.com/admin -> general admin + statistics
  • oncobol.anubex.com/help ?subject=... or ?search= or ?msg-id= 

oncobol.anubex.com/project
  • Sources: + Cobol Flavor: + [Create]
  • Status: 'no sources yet' or 'analyzing sources' or 'sources are analyzed'  + LOC quota (added + deleted)
  • Show sources: (paged) list of source files, each 'deletable' and 'clickable' + 'Add New Sources' + status: OK, parse errors + line count
  • Statistics: LOC, sources, statement counts
oncobol.anubex.com/project?source=<source-name>
  • Syntax-highlighted + grammar-linked + inter-linked source
