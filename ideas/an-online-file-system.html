Online File System

The Idea

The idea is pretty simple: have an online system that stores and provides access to a bunch of files.

Why this would be worthwhile:
  • Have access to files from anywhere
  • Possibility to implement features that current file-systems don't always provide: 
    • Full encryption
    • Custom tagging (like gmail's mail tagging)
    • Extended search facility
    • Creation, last-access and last-modified times + accessor names
    • Keep previous versions
    • File-sharing
    • Monitor all accesses/changes (via RSS feed?) to all files/specific files
  • Possibility to implement 'perfect' backup via online storage systems such as Amazon's SS3

The obvious downsides:
  • Must always download file before use (slow + bandwith usage)
  • Must always upload file after change (slow + bandwith usage)
  • No native (local, on pc) access to these files, so no easy way to use them via scripts etc.
  • No explorer-alike integration for these files

The best of both worlds would be:
  • An online file system that is authorative 
  • A local/native file system extension that emulates a local file-system on top of the 'back-end' online file system
  • Files would be cached locally and automatically synced with the online system

Design


3 phases:
  1. Simple storage + upload/download features
  2. Local file-system extensions; file-versioning; file-encryption; search by content
  3. Enable conflict-detection when updates from multiple locations can occur; maybe implement locking mechanism; solve the 'multiple-location' issue

Phase 1


Features to be implemented:
  • Upload individual files
  • Upload group of files (via zip)
  • List all files
  • Search for files by name
  • Search for files by access, creation and mod-time or name
  • Search for files by tag
  • Add custom tags to files
  • Download individual file
  • Download group of files
  • View (online) a single file assuming it is text

Phase 2


Features to be implemented:
  • Windows file-system hook that acts as a mapped drive and offers access to all files in the online file system

  • Additional online features:
    • Create account / delete account
    • Specify file preferences: case sensitive, store original paths, ...
    • Keep multiple file versions
    • Specify how many versions to keep
    • Delete file or files -> tag 'deleted'
    • Forever delete files marked as 'deleted'
